Tahapan Instalasi ekstensi:

  1. Download, lalu Extract file tersebut terlebih dahulu
  2. Masuk ke extension chrome melalui browser. Copy: chrome://extensions/
  3. Aktifkan developer mode (sudut kanan atas) kemudian Klik load unpacked (kiri atas)
  4. Pilih folder yang di extract (Netflix Premium), klik select folder
  5. Pin extension, Masuk ke Netflix.com, Klik extensionnya, klik Masuk
  6. Happy Watching
